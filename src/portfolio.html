Bienvenido a mi Portfolio

¡Hola! Soy Diego, un científico de datos apasionado por el análisis de datos y la visualización de información. En este sitio web encontrarás información sobre mis proyectos y habilidades.

Proyectos Destacados

Proyecto 1: Análisis de Ventas

Desarrollé un sistema de análisis de ventas que utiliza técnicas de minería de datos para identificar tendencias y patrones en el comportamiento de compra de los clientes, ayudando a optimizar las estrategias de marketing.

Proyecto 2: Predicción de Demanda

Este proyecto implementa modelos de aprendizaje automático para predecir la demanda de productos en función de datos históricos, lo que permite a las empresas gestionar mejor su inventario y reducir costos.

Proyecto 3: Visualización de Datos de Salud

Creé una plataforma interactiva que visualiza datos de salud pública, permitiendo a los usuarios explorar estadísticas sobre enfermedades y factores de riesgo, facilitando la toma de decisiones informadas.
